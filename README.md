# dgt-net
Thin network layer on top of Node.js, managed TCP connection & buffering. Packet writer & reader.

## Install
    npm install dgt-net

## Usage
    let server = require('dgt-net').server
    let client = require('dgt-net').client

## Basic
`dgt-net`

## Demo
